# CM_2026
The github for Computation Musicology

This repository contains the code and resources for the Computation Musicology course (CM_2026). The course covers various topics in computational musicology, including music analysis, music information retrieval, and machine learning applications in music.

The portfolio is based on my playlist "Legends", for which I will try to answer the following answers:

```text
1. How has the sound of the tracks I add to my playlist changed over time (e.g., 2016–2026), in terms of energy, danceability, valence, acousticness, and tempo?
2. Do I add “older” songs (released long before I added them) that sound measurably different from contemporary songs (released near the time I added them)?
3. Is my playlist best described as one consistent sound, or as multiple distinct clusters (e.g., EDM vs rock vs rap), and how have those clusters changed in proportion over time?
```
